# TVBOX - 在线视频播放器

一个基于HTML+CSS+JavaScript构建的TVBOX风格在线视频播放网站，包含完整的视频源管理和后台系统。

## 功能特性

### 前台功能
- 📺 美观的首页界面
- 🎬 视频源管理（添加、删除、启用禁用）
- 🎯 视频播放器（支持多源切换）
- 🔍 视频搜索功能
- 📱 响应式设计，支持各种设备
- 💰 金币统计系统

### 后台管理
- 🔐 用户登录系统（用户名：admin，密码：longben314）
- 📊 数据统计概览
- 🎥 视频源管理（表格形式）
- 📜 播放历史查看和管理
- ⚙️ 系统设置（密码修改）

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式和响应式设计
- **JavaScript** - 业务逻辑
- **LocalStorage** - 数据存储
- **无外部依赖** - 纯原生实现

## 快速开始

### 本地运行

1. 克隆项目到本地
2. 使用浏览器打开 `index.html`
3. 即可开始使用

### GitHub Pages 部署

项目已配置为支持GitHub Pages部署，部署方式：

1. 将代码推送到GitHub仓库
2. 在仓库设置 -> Pages中启用
3. 访问部署地址即可

## 使用说明

### 前台
1. 访问首页，点击"立即使用"开始
2. 在"源管理"中添加/管理视频源
3. 在"播放器"中选择源并搜索视频
4. 点击视频开始播放

### 后台
1. 点击首页"管理后台"按钮
2. 输入用户名和密码登录
3. 在概览页查看统计数据
4. 在各页面管理相应功能

## 数据存储

所有数据存储在浏览器的LocalStorage中，包括：
- 视频源列表
- 播放历史
- 用户登录状态
- 系统设置

## 开发说明

这是一个纯前端项目，没有后端服务器。所有功能都在浏览器端实现。

### 核心模块

- `SourceManager` - 视频源管理
- `AuthManager` - 认证管理
- `HistoryManager` - 播放历史管理
- `getData/saveData` - 本地存储封装
